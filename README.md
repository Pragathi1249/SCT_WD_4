# 📋 Productivity Dashboard - To-Do & Notes App

A modern, feature-rich Productivity Dashboard built using **HTML, CSS, and JavaScript** in a single file. This application helps users manage tasks, track productivity, organize notes, monitor progress, and stay on top of deadlines—all without requiring a backend or database.

## 🚀 Features

### 📊 Dashboard

* Real-time task statistics
* Total Tasks Counter
* Completed Tasks Counter
* Pending Tasks Counter
* Overdue Tasks Counter
* Today's Tasks Counter
* Upcoming Tasks Counter
* Completion Percentage Progress Bar

### ✅ Task Management

* Add, Edit, and Delete Tasks
* Mark Tasks as Completed
* Undo Completed Tasks
* Archive Tasks
* Duplicate Tasks
* Set Due Date and Time
* Priority Levels (High, Medium, Low)
* Categories (Personal, Work, Study, Health, Other)
* Tags Support
* Drag-and-Drop Task Reordering

### 📝 Notes Management

* Create Notes
* Edit Notes
* Delete Notes
* Auto Save Notes
* Instant Note Search
* Local Storage Support

### 📅 Calendar & Scheduling

* Calendar View
* Due Date Tracking
* Upcoming Tasks Display
* Overdue Task Alerts
* Deadline Countdown Timer

### 🔍 Search & Filters

* Search Tasks by Title or Description
* Filter by Status
* Filter by Priority
* Filter by Category
* Sort by Date, Priority, Name, or Status

### 📈 Analytics

* Completed vs Pending Tasks Chart
* Priority Distribution Chart
* Weekly Productivity Trend

### 🔔 Productivity Tools

* Browser Notifications
* Daily Productivity Summary
* Productivity Streak Counter
* Achievement Badges
* Motivational Quotes

### 💾 Data Persistence

* Local Storage Support
* Automatic Data Saving
* Theme Preference Saving
* Restore Data on Reload

### 🎨 User Interface

* Responsive Design
* Glassmorphism Cards
* Smooth Animations
* Light/Dark Mode
* Toast Notifications
* Confirmation Modals

### 📤 Additional Features

* Export Tasks as JSON
* Import Tasks from JSON
* Print Task Reports
* Clear Completed Tasks
* Keyboard Shortcuts

## ▶️ Getting Started

1. Download or clone the repository.

```bash
git clone https://github.com/your-username/productivity-dashboard.git
```

2. Open the project folder.

3. Double-click `index.html` or open it in any modern browser.

No installation or setup is required.
